## Manual Installation
1. Download `ServerNetworkPanel.java`, `build_panel.bat`, `compile_panel.bat`, `run_panel.bat`, and `config.properties`
2. Place them in your Network's folder
3. Configure the panel in `config.properties`
4. Run `compile_panel.bat`
5. Run `build_panel.bat`

## You have now successfully installed the panel! To run it, run `run_panel.bat`
